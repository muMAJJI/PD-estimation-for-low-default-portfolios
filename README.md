## 📌 Overview

Probability of Default (PD) is one of the most important parameters in credit risk modeling. A reliable estimation of this parameter is only possible in the case of large and sufficiently granular portfolios.

However, traditional PD estimation methods prove to be limited and sometimes inconsistent when the portfolio exhibits a very low number of defaults, or even none at all — particularly in high rating grades.

The method proposed by Dirk Tasche and Pluto introduces an innovative approach for estimating PD at given confidence levels, specifically designed for low default portfolios.

---

## 🎯 Project Objective

This project implements the Pluto & Tasche methodology for PD estimation under different portfolio configurations, with a focus on low default portfolios.

---

## ⚙️ Considered Configurations

The implementation covers the following cases:

- **Independence of default events with no observed defaults**
- **Independence of default events with a small number of defaults**
- **Correlation of default events with no observed defaults**
- **Correlation of default events with a small number of defaults**
